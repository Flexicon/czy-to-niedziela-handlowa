# Czy to niedziela handlowa?

Polish web utility that tells you whether the nearest Sunday is a commercial (trading) Sunday under Polish retail law.

## Developing

```bash
npm run dev
```

## Building

```bash
npm run build
npm run preview
```

## Testing

```bash
npm run test        # watch mode
npm run test:run    # run once
```

## Tech Stack

- Astro 6.x
- TypeScript
- date-fns
- Water.css (dark theme)
- Vitest
- Netlify (static deployment)
